## вҡҷпёҸ Installation

### 1пёҸвғЈ Clone Repository

```bash
git clone https://github.com/your-username/expense-tracker.git
cd expense-tracker
```

### 2пёҸвғЈ Backend Setup

```bash
cd backend
npm install
```

Create `.env`

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

Run Backend

```bash
npm start
```

or

```bash
npm run dev
```
